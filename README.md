# UTS Pemweb Kel 7

Simple CRUD PHP web app for managing Program Studi and Users (Bootstrap-based UI).

## Requirements
- XAMPP (Apache + MySQL) on Windows
- PHP 7.x or 8.x
- Browser (Chrome/Edge/Firefox)

## Setup
1. Place project folder in XAMPP `htdocs` (already at `c:\xampp\htdocs\uts_pemweb_kel_7`).
2. Start Apache and MySQL from XAMPP Control Panel.
3. Create the database `pemweb_db` and import SQL (if you have dump files):

   - Open `http://localhost/phpmyadmin` and create database `pemweb_db`.
   - Import any provided `.sql` files (e.g. `user_tbl.sql`) into `pemweb_db`.

4. Open the app pages in your browser:

   - Home / Login: `http://localhost/uts_pemweb_kel_7/index.php`
   - Program Studi: `http://localhost/uts_pemweb_kel_7/program_studi.php`
   - Users: `http://localhost/uts_pemweb_kel_7/user.php`
   - Dashboard: `http://localhost/uts_pemweb_kel_7/dashboard.php`
